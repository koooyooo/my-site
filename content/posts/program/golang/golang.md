## Hello Golang
Golangを使い始めてからしばらくが経ちました。当初はその単純すぎる言語仕様からGolangのことを**詰まらない言語**だと思っていました。利用し始めて2年程経ちましたが、その思いは今も変わりません。直前に利用していた言語が Kotlinや Scalaだったので、その言語仕様の差は歴然です。

```scala
object Main {
  def main(args: Array[String]) {
    println("Hello Scala")
  }
}
```

```golang
func main() {
  fmt.Println("Hello Golang")
}
```
